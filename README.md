<h1>🚀 PokeMAKER</h1>
<p>Interfaz web diseñada para la gestión integral de fangames de Pokémon. Olvidate de los Excels y ten un control total y visual de tus proyectos en un solo lugar.</p>
<p><em>Web interface designed for the comprehensive management of Pokémon fangames. Forget about Excels and have total, visual control of your projects in one place.</em></p>

<h2>🔗 Acceso a la App / Access the App</h2>
<p><a href="https://krislaz07.github.io/PokeMaker/"><strong>¡Probalo ahora aquí! / Try it out here!</strong></a></p>

<h2>🛠️ Funciones Principales</h2>
<ul>
    <li><strong>DEX de Encuentros:</strong> Gestioná Pokémon, rutas, probabilidades y tipos de encuentro. Exportá tus listas a formato .txt.</li>
    <li><strong>Rutas:</strong> Visualizador de rutas con análisis de tipos y soporte para objetos.</li>
    <li><strong>Objetos:</strong> Base de datos centralizada de items con asignación de ubicación.</li>
    <li><strong>Personajes:</strong> Fichas detalladas para NPCs, facciones, roles y equipos Pokémon.</li>
    <li><strong>Tareas:</strong> Tracker de desarrollo para categorizar pendientes por tipo y urgencia.</li>
    <li><strong>Guion:</strong> Espacio creativo para redactar y organizar escenas y eventos.</li>
    <li><strong>Configuración:</strong> Personalización de rutas, facciones y clases. Incluye sistema <strong>Backup Master</strong> para exportar e importar toda tu información de forma segura.</li>
</ul>

<h2>🛠️ Main Features</h2>
<ul>
    <li><strong>Encounter DEX:</strong> Manage Pokémon, routes, probabilities, and encounter types. Export your lists to .txt format.</li>
    <li><strong>Routes:</strong> Route visualizer with type analysis and item support.</li>
    <li><strong>Items:</strong> Centralized item database with location assignment.</li>
    <li><strong>Characters:</strong> Detailed sheets for NPCs, factions, roles, and Pokémon teams.</li>
    <li><strong>Tasks:</strong> Development tracker to categorize pending items by type and urgency.</li>
    <li><strong>Script:</strong> Creative space to write and organize scenes and events.</li>
    <li><strong>Settings:</strong> Customize routes, factions, and classes. Includes <strong>Backup Master</strong> system to export and import all your information securely.</li>
</ul>

<h2>💾 Sobre los datos / About your data</h2>
<p>PokeMAKER guarda toda tu información <strong>localmente en tu navegador</strong>.</p>
<p><em>PokeMAKER saves all your information <strong>locally in your browser</strong>.</em></p>
<blockquote>
    <p><strong>IMPORTANTE:</strong> No olvides utilizar la función de "Descargar Backup Completo" en el apartado de Configuración periódicamente para guardar una copia de seguridad en tu computadora y evitar pérdidas de datos.</p>
    <p><em><strong>IMPORTANT:</strong> Don't forget to use the "Descargar Backup Completo" feature in the Settings tab periodically to save a backup on your computer and avoid data loss.</em></p>
</blockquote>
